# starter_for_3d_minigames
微信小游戏 starter（for 3D 游戏）。提供了 Web adaptor，跑通了 Babylon.js、Three.js。

使用小游戏模拟器打开本工程即可使用。默认使用 Babylon.js，可在 game.js 里切换为 Three.js。
